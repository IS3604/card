# 💍 Anniversary Card

A beautiful interactive anniversary card built with HTML, CSS, and JavaScript.

## How to View

Open `index.html` in any browser — or host it with **GitHub Pages**:

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your card will be live at `https://<your-username>.github.io/<repo-name>/`
